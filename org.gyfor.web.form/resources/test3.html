Forms

Checkboxes & Radio

Kube features full variety of stylish checkboxes and radio buttons. You can feature them on your pages by defining input type as checkbox or radio. There's also a neat option to place checkboxes inline by adding checkbox-inline class to the container (works for both checkboxes and radio buttons)

<form method="post" action="" class="form">
    <div class="form-item checkboxes">
        <label><input type="checkbox"> Check 1</label>
        ...
    </div>
</form>

<form method="post" action="" class="form">
    <div class="form-item checkboxes-inline">
        <label><input type="checkbox"> Check 5</label>
        ...
    </div>
</form>

<form method="post" action="" class="form">
    <div class="form-item checkboxes">
        <label><input type="radio"> Check 1</label>
        ...
    </div>
</form>

<form method="post" action="" class="form">
    <div class="form-item checkboxes-inline">
        <label><input type="radio"> Check 5</label>
        ...
    </div>
</form>

Fieldset

Fieldsets in Kube are nicely pre-formatted and ready to go. Just wrap your form in a fieldset tag and you're good to go.

Login data
About 
<form method="post" action="" class="form">
    <fieldset>

        <legend>Login data</legend>

        <div class="form-item">
            <label>Email</label>
            <input type="email" name="user-email" class="width-50">
        </div>

        <div class="form-item end">
            <label>Password</label>
            <input type="password" name="user-password" class="width-50">
        </div>

    </fieldset>

    <fieldset>
        <legend>About</legend>

        <div class="form-item">
            <textarea name="user-about" rows="5"></textarea>
        </div>

        <div class="form-item end">
            <button>Submit</button>
        </div>

    </fieldset>
</form>

Big

Simple class big makes your selects and fields, well, bigger.

<form class="form">
    <div class="form-item">
        <input type="text" class="big" placeholder="Title">
    </div>
</form>

<form class="form">
    <div class="form-item">
        <select class="big">
            <option>...</option>
        </select>
    </div>
</form>

Width

Following framework-wide rules, it is very easy to manipulate form fields sizes. Just use width-50 class to make a field 50% wide.

<form method="post" action="" class="form">
    <div class="form-item">
        <input type="text" class="width-50">
    </div>
</form>

States

By default, Kube features two different states: success and failure. 

<form method="post" action="" class="form">
    <div class="form-item">
        <label>Label <span class="success">...</span></label>
        <input type="text" class="success">
    </div>
</form>

<form method="post" action="" class="form">
    <div class="form-item">
        <label>Label <span class="error">...</span></label>
        <input type="text" class="error">
    </div>
</form>

Required

Along with making a field actually required, you can add a visual clue for the users using a span with req class.

<form method="post" action="" class="form">
    <div class="form-item">
        <label>Email <span class="req">*</span></label>
        <input type="email" name="user-email" class="width-50">
    </div>
</form>

Descriptions

Descriptions are simple: as long as form's element has desc class, Kube will treat it as a description.

Please enter your email.
<form method="post" action="" class="form">

    <div class="form-item">
        <label>Name <span class="desc">...</span></label>
        <input type="text" name="user-name" class="width-50">
    </div>

    <div class="form-item">
        <label>Email</label>
        <input type="email" name="user-email" class="width-50">
        <div class="desc">...</div>
    </div>
</form>

Disabled

Not only text inputs can be disabled, but checkboxes and radio buttons as well. Just add disabled attribute to the input.


<input type="text" value="" disabled>
<input type="checkbox" disabled>
<input type="radio" disabled>
<textarea disabled></textarea>
<select disabled>
    <option>...</option>
</select>

Append & Prepend

You can prepend or append certain elements within your input fields, such as currency characters.

$ 
<form method="post" action="" class="form">
    <div class="form-item">
        <label>How much is it?</label>
        <div class="controls width-50">
            <span>$</span>
            <input type="text">
        </div>
    </div>
</form>
$ 
<form method="post" action="" class="form">
    <div class="form-item">
        <label>It is much how?</label>
        <div class="controls">
            <input type="text"><span class="append">$</span>
        </div>
    </div>
</form>

Button Append

Appending works for buttons as well. Here you can see a button that has been appended and included withing a text input field. 

<form method="post" action="" class="form">
    <div class="form-item">
        <label>Label</label>
        <div class="controls width-50">
            <input type="text" name="search" placeholder="Search">
            <button class="button primary">Go</button>
        </div>
    </div>
</form>
